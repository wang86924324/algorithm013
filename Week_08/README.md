
位运算实战要点：
**判断奇偶**
**x%2=1==>x&1=1**
例如：
1%2=1  ==>1&1=1
11%2=1 ==>11&1=1

**x%2=0==>x&1=0**
例如：
0%2=0==>0&1=0
10%2=0==>10&01=0



**x&(x-1)  110&101=100  清理最低位1**
**x&~x=0 110&001=0**  
x&(-x) ->得到最低位的1

负数表示形式：

原码、补码、反码举例：

原码：负数原码是绝对值原码最高位补1，假设机器是4位（一般是32或者64位）

6->0110

-6->11100

反码：正数原码等于反码，负数反码是其原码除符号位外，各位取反。

6->0110

-6->1001

补码：整数的补码跟原码相同，负数的补码是原码除符号位外，各位取反，最后一位+1

例如：

6->0110

-6->1010

例如：

110&-110=10
